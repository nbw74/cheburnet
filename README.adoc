= Cheburnet

----

     [Cross-border WAN]         [Domestic WAN]
             ^                        ^
             |                        |
           SNAT         GRE         SNAT    GRE over IPsec
           [R3] <------------------ [R2] <=================== [R1]
                   STATIC routes                  0/0

         Foreign                  Domestic                 Home/Office
         router                    router                    router

----

. Configure tunnels R1 <--> R2 and R2 <--> R3
. Install FRR on all routers and configure OSPF daemons (see *frr.conf* for R3 configuration example)
. Setup SNAT on R2 and R3
. On R3:
.. place *interface* and *routes* configuration files in /etc/cheburnet/
.. place *cron* file in */etc/cron.d/cheburnet*
.. place *cheburnet.sh* in /usr/local/bin/
.. run *cheburnet.sh* for setup static routes
. On R1: configure default route (0.0.0.0/0) pointing to R2 tunnel endpoint
