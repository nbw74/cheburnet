= Cheburnet

----
 [Cross-border WAN]    [Domestic WAN]
         ^                   ^
         |                   |
       SNAT      GRE       SNAT  GRE over IPsec
       [R3] <------------- [R2] <============== [R1]
              OSPF routes             0/0

     Foreign             Domestic            Home/Office
     router               router               router

----

. Configure tunnels R1 <--> R2 and R2 <--> R3
. Install FRR on R2 and R3 (optionally on R1)
. Enable OSPF daemon on routers (see *frr.conf* for R3 configuration example)
. Setup SNAT on R2 and R3
. On R3: place *interface* and *routes* configuration files in /etc/cheburnet/
. On R3: run *cheburnet.sh* for setup static routes
. On R1: configure default route (0.0.0.0/0) pointing to R2
